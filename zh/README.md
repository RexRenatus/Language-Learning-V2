# Chinese — Progressive Course + Deck Collection

> ## 🎓 Chinese Progressive Course — the recommended study path
>
> The flagship Chinese product is the **Progressive Course**: **86 units, A1→C2, 40,059 cards,
> 11,562 vocab words** (~117 MB). It opens with **4 from-zero foundation units** (pinyin + the four
> tones, numbers incl. 二/两 and 万-grouping, colours, survival phrases), then each unit is a CEFR
> can-do lesson that runs **0 Dialogue → 1 Vocab → 2 Grammar → 3 Transform → 4 Dictation**, grammar
> sequenced like a textbook (是/有 → **了 hub** → complements → HSK 4→5) with native-speaker audio
> throughout. The **C1/C2 band adds register/discourse/idiom/nuance** (grounded in HSK 6, the
> language's C1–C2 standard). Eight card types — incl. B1+ **mediation · composition · reading**, a **character-recognition** card (hanzi → reading + meaning), and **tone** + **audio→pinyin transcription** drills — **zero multiple-choice**.

## Source

- **Frequency list:** OpenSubtitles 2018 — [`words/zh-frequency-list.txt`](words/zh-frequency-list.txt) (50,000 entries).
- **Glosses:** Free dictionaries (Wiktionary / CC-CEDICT / JMdict) → Gemini fallback for the long tail.
- **Tokenization / IPA:** jieba (segmentation), pypinyin + pinyin-to-ipa (Pinyin + IPA).
- **Example sentences & reasoning:** Google Gemini 3 Flash Preview.
- **Semantic distractors:** BAAI/bge-m3 (1024-dim multilingual embedding) — Synonym deck also runs every pair through a Gemini binary verifier to filter character-overlap false positives.
- **MCQ format:** 6-option for vocab decks, 4-option for cloze decks.

## How to import

1. Install [Anki Desktop](https://apps.ankiweb.net/) (free).
2. Download the `.apkg` file you want.
3. In Anki: **File → Import** → select the file.

Each deck imports as a separate sub-deck so you can install them side-by-side without conflicts.
