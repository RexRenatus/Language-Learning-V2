# Language-Learning-V2

A **structured A1ΓÁÈnative language course** for **Spanish, French, Japanese, Chinese (Mandarin),
and Korean** ΓÇõ study units in order, like a textbook, with native-speaker audio throughout.

The **Progressive Course** is the flagship: study **units in order**, each one a CEFR can-do
lesson built from **eight card types and zero multiple-choice** ΓÇõ every card tests *active*
knowledge (production, listening, dictation, mediation, writing, reading). The 10 type-decks further down
(120,890 cards) are now the **content pool** the course is assembled from ΓÇõ still downloadable
as reference/drill material, but no longer the headline study path.

## ≡ÓÃô Progressive Course ΓÇõ the flagship

A **structured A1ΓÁÈnative course**, not just flashcards. Study **units in order**. Each course opens
with **4 from-zero foundation units** ΓÇõ writing system & pronunciation, numbers, colours, and
days/time & survival phrases ΓÇõ where letter/sound cards show **IPA** and play **phoneme-forced

> **0 Dialogue** (comprehensible input ┬╖ audio) ΓÁÈ **1 Vocab** (type the word ┬╖ listen) ΓÁÈ

> **5 Mediation** (relay/summarise a source for a third party) ΓÁÈ **6 Composition** (genre-scaffolded
> guided writing ΓÁÈ model + self-check) ΓÁÈ **7 Reading** (authentic-style passage ΓÁÈ literal + inferential questions)

**Eight card types, zero multiple-choice** ΓÇõ every card tests **active knowledge** (production,
listening, dictation, mediation, free writing, reading comprehension), with **native-speaker neural audio** (Google) throughout.

**How it works.** *Decks* ΓÇõ import the `.apkg` into **AnkiMobile** (iPad): the front shows the reading
+ meaning (and plays the audio), you trace the character, and each stroke is graded with a hint after
two misses; the back reveals a numbered stroke-order diagram. *Workbooks* ΓÇõ open the PDF in **GoodNotes**
and write in the squared grids; each is ordered into **Sessions** that introduce a few new characters
and re-present earlier ones at expanding intervals (+1/+2/+4/+8/+16 sessions).

> **On cursive.** No freely-licensed *animated* cursive dataset exists, so cursive isn't stroke-graded.
> Instead each cursive **deck** card gives a **freehand trace canvas** ΓÇõ the cursive model glyph (zh ΦíÔΣ╣ª
> ┬╖ ja ΦíÔµ¢╕ ┬╖ ko φÙÌδª╝∞▓┤, OFL fonts) shows faint and you trace it with finger / Apple Pencil, with a
> **ΓÚ╢ Stroke order** button that animates the underlying regular skeleton; the **workbooks** print the
> model over a faint skeleton to copy. A handful (~30) of rare Japanese kanji ship cursive-only.

### ≡ÓÔÉ Global decks ΓÇõ one file, all languages, level-tiered

Each global deck is a single `.apkg` with a 3-tier nested structure:

```
{Type}.apkg
  ΓõõΓõÇΓõÇ {Type}::Spanish
        Γõ£ΓõÇΓõÇ {Type}::Spanish::A1   ΓÁÉ cards sorted by frequency rank (most common first)
        Γõ£ΓõÇΓõÇ {Type}::Spanish::A2
```

Inside each level subdeck, cards are sorted **by frequency** ΓÇõ top-1000 words first, then 1K-3K, etc. Anki's review order respects this, so you learn high-frequency / lower-CEFR cards before rare / higher-CEFR ones automatically.

**Pronunciation:** Every cloze-MCQ and Numbers card now carries IPA (International Phonetic Alphabet) transcriptions in the original ebook format ΓÇõ full sentence in the prompt panel, answer-word repeated below the green Γ£ô banner. Generated via `espeak-ng + epitran` (Spanish), `pykakasi + pyopenjtalk` (Japanese), `pypinyin + pinyin_to_ipa` (Chinese), and rule-based HangulΓÁÈIPA (Korean). Format: `/word | word | wordΓÁÌ ΓÇÚ/` with `|` for word boundaries, `ΓÁÌ ΓÇÚ` for sentence-final declaratives.

### ≡ÓôÜ Studying the raw pool decks (advanced / self-directed)
If you'd rather drill the type-decks directly instead of following the
[course](#-progressive-course--the-flagship), they're **parallel CEFR-gated tracks**, not
sequential blocks. Function words aren't gated ΓÇõ they're the highest-frequency words (~50% of
all text), so learn them **from day one**. Grammar and conjugation need a vocabulary base to
attach to, so they unlock once you hold the high-frequency core:

1. **Start together:** Function-Words ┬╖ Vocab-MCQ (A1) ┬╖ Numbers (1ΓÇô20).
2. **At ~the top 1,000 words (A1 vocab tier):** add **Conjugation A1** + **Grammar A1**.
3. **A2 vocab** ΓÁÈ Conjugation A2 ┬╖ Grammar A2 ┬╖ Counters.
4. **B1+ vocab** ΓÁÈ Synonyms ┬╖ Sentence-Mining ┬╖ Register.

Rule of thumb: **~300ΓÇô500 words to *begin* grammar/conjugation, ~1,000 (the A1 tier) to
progress comfortably** (ΓÊê80ΓÇô85% spoken-text coverage). Full policy + per-language
thresholds + deck-unlock matrix: `Phase 11-Global-Decks/STUDY-ORDER.md`.

## What's in each deck type

- **Classic vocab card** ΓÇõ Word + IPA (and Pinyin for Chinese) on the front; English meaning, example sentence, top collocations on the back.
- **MCQ Recognition** ΓÇõ English definition shown; user picks the matching target-language word out of six options.
- **MCQ Recall** ΓÇõ Target-language word + IPA shown; user picks the matching English definition out of six options.
  - _Both MCQ decks show a **color-coded example sentence** on the back: each target-language word and its English translation share a color (grammatical/function words are gray), so word order and grammar ΓÇõ e.g. SOVΓÁÈSVO reordering, particles, negation ΓÇõ are visible at a glance._
  - _**Spoken pronunciation ≡Óõè** ΓÇõ every headword carries a native-speaker **male voice** clip whose pronunciation is **phoneme-forced** from the data already on the card (IPA for Spanish/French, kana for Japanese, pinyin for Chinese; Korean reads its phonetic Hangul) ΓÇõ so even single-letter function words say the word, not the letter-name. Generated with Google Cloud TTS neural voices, loudness-equalized, and **autoplaying three times** where the word is the focus: the **front of Recall** cards and the **back of Recognition** cards._
  - _**Japanese furigana** ΓÇõ kanji headwords show their **kana reading** beside the word (Φ╗è ΓÁÈ πüÂπéÍπü╛), with the reading, IPA, and audio all derived from one analyzer (pyopenjtalk) so they always agree._
- **Function-Word Cloze** ΓÇõ A sentence with a function word blanked. 4 options with one correct + 3 sentence-aware distractors. Back reveals the answer + a one-sentence rule explanation.
- **Grammar-Pattern Cloze** ΓÇõ A sentence drilling a specific grammar pattern (`πÇ£πüÓπüã`, `comparativos`, `VΣ╕ìV`, `πÇ£∞ÙÇ ∞áü∞Ù┤ ∞₧ê∞Ú┤∞Üõ`, etc.). Distractors are *typed* (tense / confusable / register variant).
- **Sentence-Mining Cloze** ΓÇõ Top-frequency vocab word blanked inside its real-corpus example sentence. 4-option MCQ from the same word's pre-built distractor pool.
- **Conjugation Drill** ΓÇõ A sentence requiring the correct tense/aspect/agreement form of a target verb. Distractors are *other legitimate forms of the same verb*.
- **Counter / Measure-Word** ΓÇõ Pick the right counter for the noun being quantified (`µ£¼πéÈΣ╕Ê____Φ▓╖πüãπü╛πüùπüÓ` ΓÁÈ `σÁè`).
- **Synonyms** ΓÇõ Pick the closest synonym for a target word. Every candidate pair was binary-verified by Gemini, fixing the CJK character-overlap failure mode.
- **Honorifics / Register** ΓÇõ Choose the speech-level-appropriate expression for a stated social context (formal email vs casual peer chat, etc.).

## Quality assurance

Three independent audits run on every release:

1. **Source-data audit** ΓÇõ for every card: answer Γêê options, 4 distinct options, no empty fields, answer present in sentence, blank marker present, English present, Explanation present.
2. **APKG file audit** ΓÇõ extracts the sqlite under each .apkg; verifies field count, no answer leak on the front, the correct option marked on the back.
3. **Randomization audit** ΓÇõ verifies the correct answer is uniformly distributed across A/B/C/D positions (within statistical expectation), no positional bias.

Latest release (May 2026): **0 issues, ╧ç┬▓ < 12 on every cloze-MCQ deck.**
