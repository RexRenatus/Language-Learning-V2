# Korean — Full Deck Collection

Comprehensive Korean flashcards spanning vocabulary, grammar patterns, and contextual usage. All decks built from the OpenSubtitles 2018 frequency list (top-50K) with rich enrichment.

> **▶ Status — legacy / pending QA.** The project's **active study set** is the global
> **Vocab-MCQ Recognition + Recall** decks (full A1–B2, all 5 languages incl. French) — see
> the [main README](../README.md). The per-language decks on this page (including the
> retired **Classic vocab**, SPEC-LANG-002) predate that refinement and are **pending
> quality review** — kept downloadable, but not the recommended study path.

## Source

- **Frequency list:** OpenSubtitles 2018 — [`words/ko-frequency-list.txt`](words/ko-frequency-list.txt) (50,000 entries).
- **Glosses:** Free dictionaries (Wiktionary / CC-CEDICT / JMdict) → Gemini fallback for the long tail.
- **Tokenization / IPA:** kiwipiepy (POS / morpheme analysis), rule-based Hangul→IPA.
- **Example sentences & reasoning:** Google Gemini 3 Flash Preview.
- **Semantic distractors:** BAAI/bge-m3 (1024-dim multilingual embedding) — Synonym deck also runs every pair through a Gemini binary verifier to filter character-overlap false positives.
- **MCQ format:** 6-option for vocab decks, 4-option for cloze decks.

## How to import

1. Install [Anki Desktop](https://apps.ankiweb.net/) (free).
2. Download the `.apkg` file you want.
3. In Anki: **File → Import** → select the file.

Each deck imports as a separate sub-deck so you can install them side-by-side without conflicts.
