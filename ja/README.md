# Japanese — Progressive Course + Deck Collection

> ## 🎓 Japanese Progressive Course — the recommended study path
>
> The flagship Japanese product is the **Progressive Course**: **87 units, A1→C2, 11,894 cards,
> 4,791 vocab words** (~61 MB). It opens with **4 from-zero foundation units** (kana review,
> numbers incl. native vs Sino-Japanese, colours, survival phrases), then each unit is a CEFR
> can-do lesson that runs **0 Dialogue → 1 Vocab → 2 Grammar → 3 Transform → 4 Dictation**, with
> grammar sequenced like a textbook (です → polite forms → **て-form hub** → N3 → N2) and
> native-speaker audio throughout. The **C1/C2 band** adds register/discourse/idiom/nuance
> (grounded in the JLPT N1 / C1–C2 standard). Eight card types — incl. B1+ **mediation · composition · reading** — **zero multiple-choice**.

## Source

- **Frequency list:** OpenSubtitles 2018 — [`words/ja-frequency-list-unidic.txt`](words/ja-frequency-list-unidic.txt) (42,833 UniDic dictionary forms).
- **Glosses:** Free dictionaries (Wiktionary / CC-CEDICT / JMdict) → Gemini fallback for the long tail.
- **Tokenization / IPA:** fugashi + UniDic (POS / dictionary forms), fugashi + OpenJTalk (IPA).
- **Example sentences & reasoning:** Google Gemini 3 Flash Preview.
- **Semantic distractors:** BAAI/bge-m3 (1024-dim multilingual embedding) — Synonym deck also runs every pair through a Gemini binary verifier to filter character-overlap false positives.
- **MCQ format:** 6-option for vocab decks, 4-option for cloze decks.

## How to import

1. Install [Anki Desktop](https://apps.ankiweb.net/) (free).
2. Download the `.apkg` file you want.
3. In Anki: **File → Import** → select the file.

Each deck imports as a separate sub-deck so you can install them side-by-side without conflicts.
